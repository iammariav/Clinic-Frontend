## Project Overview
The Clinic Appointment System Frontend is a web-based application developed using React with Vite.  
It allows users to manage clinic appointments by creating, viewing, updating, and deleting records.  
The frontend communicates with a RESTful backend API using Axios.

This project was developed for academic purposes as part of a software engineering requirement.

## Technologies Used
- React 18
- Vite
- Axios
- JavaScript (ES6+)
- HTML5
- CSS3
- ESLint
- Vercel (Deployment)


# Manual testing was performed for the following functionalities:
-Create appointmen
-Input validation
-View appointment list
-Update appointment details
-Delete appointment
Test cases and bug reports were documented separately.

# Known Issues:
-The frontend fails to load data if the backend API URL is set to localhost in production.
-Proper environment variable configuration is required for successful deployment.
